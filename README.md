# 🛎️ TripNest

**Lema:** *Reservá fácil, viajá mejor*  
**Tipo de aplicación:** Plataforma de reservas de alojamientos (hoteles, hostales, apartamentos, casas)

---

## 📌 Descripción

**TripNest** es una plataforma web que permite a los usuarios buscar, visualizar y reservar alojamientos de forma sencilla. También ofrece un panel de administración donde se pueden gestionar productos, usuarios, categorías y características.

Este proyecto fue desarrollado como parte de un desafío profesional siguiendo una metodología ágil en dos Sprints.

---

## 🚀 Tecnologías utilizadas

- **Backend:** Java 24 + Spring Boot
- **Frontend:** React.js
- **Base de datos:** MySQL
- **Seguridad:** Spring Security + JWT
- **ORM:** JPA / Hibernate
- **Correo:** JavaMailSender (notificaciones)
- **Control de versiones:** Git + GitHub

---

## 🧩 Funcionalidades principales

### 🧾 Sprint 1

- Estructura básica del sitio: header, footer y layout general.
- CRUD de productos (alojamientos) desde el panel de administración.
- Visualización de productos aleatorios en el home.
- Vista de detalle con galería de imágenes.
- Paginación de productos.
- Diseño responsive adaptado a desktop, tablet y mobile.

### 🔐 Sprint 2

- Registro de usuario con validaciones.
- Inicio y cierre de sesión.
- Roles de usuario y administrador.
- Asociación de categorías y características a los productos.
- Visualización de características con íconos.
- Filtros por categoría (en progreso).
- Notificaciones automáticas por email:
  - Confirmación de registro
  - Recuperación de contraseña
  - Promoción a administrador
  - Bloqueo de cuenta
  - Eliminación de cuenta
